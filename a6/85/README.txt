README for attackers
*********************

Run the executable provided to you

The application will ask if you want to compute the sum function or compute the hidden function. Enter 1 to compute sum and enter 2 to compute the hidden function.

After this, the application will ask for two numbers as input.

If you had chosen to compute the hidden function the application will ask for a password. If you enter the correct password the result will displayed, otherwise "Unauthorized user" shall be displayed.


If you had chosen the sum function the result of the sum should be displayed without any password.
